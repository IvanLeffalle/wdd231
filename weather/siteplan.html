Weather Dashboard

An Individual Project for WDD231 by Ivan Leffalle

Site Purpose

This website serves as a weather dashboard designed to provide users with real-time weather information for any city they search. By connecting to an external API like OpenWeatherMap, the site displays current weather conditions along with a five-day forecast. Key details such as temperature, humidity, wind speed, and overall weather conditions are presented in an easy-to-read format. The dashboard aims to be both functional and user-friendly, demonstrating the application of modern web development techniques.

The project highlights the integration of dynamic content rendering, interactive user features, and responsive design principles. It is a practical and relevant tool, offering valuable weather insights to users while showcasing essential skills in HTML, CSS, JavaScript, and API utilization.

Scenarios

How can I check the weather forecast for my upcoming trip?

Visitors may want to plan their travel by checking the weather for their destination. The site will provide current weather data along with a five-day forecast to help users plan ahead.

What are the weather conditions like in my city right now?

Users may want to get real-time weather information for their current location. The site will allow users to search for their city and view the current temperature, humidity, wind speed, and more.
